## 录制设置

### 通用设置

#### 分辨率

- **推荐**: 1920x1080 (1080p)
- **最低**: 1280x720 (720p)
- **说明**: 高分辨率提供更好的观看体验

#### 帧率

- **推荐**: 30 fps
- **演示/教程**: 30 fps
- **游戏录制**: 60 fps
- **说明**: 30 fps 足够用于教程和演示

#### 音频设置

- **采样率**: 44.1 kHz
- **声道**: 单声道（旁白）或立体声（背景音乐）
- **比特率**: 128 Kbps
- **说明**: 清晰的音频是关键

#### 编码器

- **推荐**: x264 (H.264)
- **备选**: H.265 (HEVC)
- **说明**: H.264 兼容性最好

#### 比特率

- **1080p @ 30fps**: 2500-5000 Kbps
- **720p @ 30fps**: 1500-3000 Kbps
- **说明**: 根据内容复杂度调整

### Windows 录制

#### 使用 Windows 内置录制

```powershell
# 使用 Xbox Game Bar
# 快捷键: Win + G
# 开始录制: Win + Alt + R
```

#### 使用 PowerShell 脚本

```powershell
# 自动化录制脚本
$recordingTime = 300  # 5 分钟

Start-Process "C:\Program Files\OBS\obs64.exe" -ArgumentList "--startrecording"

Start-Sleep -Seconds $recordingTime

Start-Process "C:\Program Files\OBS\obs64.exe" -ArgumentList "--stoprecording"
```
