## 文档模板

### API 参考文档模板

```markdown
# API Reference

## File Operations

### read_file

读取文件内容。

**参数**:

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| absolute_path | string | 是 | 文件的绝对路径 |
| offset | number | 否 | 起始行号（0-based）|
| limit | number | 否 | 读取的行数 |

**返回值**:

文件内容字符串。

**示例**:

\`\`\`javascript
read_file({
  absolute_path: "/path/to/file.txt",
  offset: 0,
  limit: 100
});
\`\`\`

**注意事项**:

- 必须使用绝对路径
- 不支持相对路径
- offset 和 limit 必须同时使用
```

### 故障排除指南模板

```markdown
# Troubleshooting Guide

## 安装问题

### 问题：技能无法调用

**症状**:

调用技能时失败，提示技能不存在。

**可能原因**:

1. 技能未安装
2. 技能列表未刷新
3. 技能文件路径不正确

**解决步骤**:

1. 检查技能是否已安装：
   \`\`\`bash
   /skills list
   \`\`\`

2. 如果技能不存在，安装技能：
   \`\`\`bash
   cp -r skills/* ~/.iflow/skills/
   \`\`\`

3. 刷新技能列表：
   \`\`\`bash
   /skills refresh
   \`\`\`

**预防措施**:

- 安装技能后立即刷新技能列表
- 定期检查技能列表

**相关资源**:

- [集成指南](INTEGRATION.md)
- [技能索引](../lib/SKILL-INDEX.md)
```
